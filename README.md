# Installation de l'extension non empaquetée sur Chrome

Ce guide vous explique comment installer cette extension Chrome en mode développeur, sans passer par le Chrome Web Store.

## Prérequis

- Google Chrome installé sur votre ordinateur
- Les fichiers de l’extension présents sur votre machine (téléchargez ou clonez ce dépôt)

## Étapes d’installation

1. **Ouvrez Chrome**  
   Lancez votre navigateur Google Chrome.

2. **Accédez à la page des extensions**  
   Dans la barre d’adresse, saisissez :  
   ```
   chrome://extensions
   ```
   puis appuyez sur Entrée.

3. **Activez le mode développeur**  
   En haut à droite de la page, activez le bouton « Mode développeur ».

4. **Chargez l’extension non empaquetée**  
   Cliquez sur le bouton « Charger l’extension non empaquetée ».

5. **Sélectionnez le dossier de l’extension**  
   Dans la fenêtre qui s’ouvre, naviguez jusqu’au dossier contenant les fichiers de l’extension (là où se trouve le fichier `manifest.json`), puis cliquez sur « Sélectionner le dossier ».

6. **L’extension est installée !**  
   Elle devrait maintenant apparaître dans la liste de vos extensions et être utilisable.

---

**Remarque :**  
Si vous modifiez le code de l’extension, vous pouvez cliquer sur le bouton « Actualiser » (icône en forme de flèche circulaire) sur la page des extensions pour recharger la version modifiée.

Bonne utilisation !
